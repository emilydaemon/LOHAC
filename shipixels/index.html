Import Custom Sprites (experimental)

HOW TO IMPORT/UPLOAD FILES

You will need a link to your image. Feel free to use a hosting service like File Garden.

Once you upload your image, copy the link to it and put it in one of the input textboxes.

How to format custom characters

All character assets should face right. Both sides use the same set of images and do not have to be flipped manually.

Each image is 70px in height, with the width being up to you. The character's feet should start on the very bottom, and any excess space at the top should not be removed. Canvas does not allow you to change the pivot of images that are inserted, so otherwise your character is just gonna float.

How to format custom romance symbols

Custom romance images are the same size as the canvas (77x70, in pixels). Whatever goes in this image is up to you.

Keep in mind the symbol is always kept on top of the canvas!
