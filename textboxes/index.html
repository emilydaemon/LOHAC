The talksprite should be a static image that is the height of the canvas (508px).

You will need to upload the sprite externally. Feel free to use a site like FileGarden!

Formatting Talksprites

As this is a HTML5 Canvas tool, you can use any color formatting supported by HTML.

To keep things simple, you may select from these common formats:

  • RGB and RGBA values (formatted rgb(value) and rgba(value) respectively)
  • Hexadecimal/hex codes (make sure the # is at the front)
  • The 140 preset CSS color names
